# Σκοπός της σειράς

Στόχος της σειράς είναι να οδηγήσει έναν αρχάριο, χωρίς καμία προηγούμενη γνώση, στη δημιουργία και δημοσίευση μιας ολοκληρωμένης δυναμικής εφαρμογής χρησιμοποιώντας σύγχρονες τεχνολογίες Web Development.

Η σειρά δεν επικεντρώνεται μόνο στη σύνταξη του κώδικα, αλλά και στην ανάπτυξη προγραμματιστικής σκέψης, στην κατανόηση του τρόπου λειτουργίας του Web και στην ικανότητα μελέτης και ανάλυσης οποιουδήποτε κώδικα.

# ΕΝΟΤΗΤΑ 2 - Περιβάλλον Ανάπτυξης

## 2.1 - Εγκατάσταση XAMPP

Σε αυτό το μάθημα θα εγκαταστήσουμε το XAMPP και θα δημιουργήσουμε το πρώτο τοπικό περιβάλλον ανάπτυξης. Θα γνωρίσουμε τα βασικά στοιχεία του πακέτου και θα προετοιμάσουμε τον υπολογιστή μας ώστε να μπορεί να εκτελεί εφαρμογές PHP χωρίς σύνδεση στο Internet.

## 2.2 - Apache

Θα γνωρίσουμε τον Apache Web Server και θα κατανοήσουμε τον ρόλο του στη λειτουργία μιας ιστοσελίδας. Θα δούμε πώς εξυπηρετεί τα αιτήματα των χρηστών, πώς συνεργάζεται με την PHP και ποιες είναι οι βασικές του ρυθμίσεις.

## 2.3 - PHP

Θα εγκαταστήσουμε και θα ελέγξουμε ότι η PHP λειτουργεί σωστά στο τοπικό μας περιβάλλον. Παράλληλα θα εξηγήσουμε τι ακριβώς είναι η PHP, πώς εκτελείται στον διακομιστή και γιατί αποτελεί μία από τις δημοφιλέστερες γλώσσες ανάπτυξης δυναμικών ιστοσελίδων.

## 2.4 - MySQL

Σε αυτό το μάθημα θα γνωρίσουμε τη MySQL και τον ρόλο των βάσεων δεδομένων στις δυναμικές εφαρμογές. Θα δημιουργήσουμε την πρώτη μας βάση δεδομένων και θα δούμε πώς αποθηκεύονται και οργανώνονται τα δεδομένα.

## 2.5 - VS Code

Θα εγκαταστήσουμε και θα ρυθμίσουμε το Visual Studio Code, τον βασικό επεξεργαστή κώδικα που θα χρησιμοποιούμε σε όλη τη σειρά. Θα γνωρίσουμε το περιβάλλον εργασίας και τις δυνατότητες που προσφέρει για γρήγορη και αποδοτική ανάπτυξη εφαρμογών.

## 2.6 - Extensions

Θα εγκαταστήσουμε χρήσιμες επεκτάσεις που βελτιώνουν σημαντικά την εμπειρία προγραμματισμού. Θα δούμε ποιες είναι πραγματικά απαραίτητες, πώς μας βοηθούν στην ανάπτυξη και ποιες αξίζει να αποφεύγουμε.

## 2.7 - Git

Σε αυτό το μάθημα θα γνωρίσουμε το Git, το πιο διαδεδομένο σύστημα ελέγχου εκδόσεων. Θα μάθουμε γιατί είναι απαραίτητο σε κάθε έργο λογισμικού και πώς μας επιτρέπει να παρακολουθούμε κάθε αλλαγή στον κώδικά μας.

## 2.8 - GitHub Desktop

Θα χρησιμοποιήσουμε το GitHub Desktop για να διαχειριζόμαστε εύκολα τα repositories μας χωρίς τη χρήση της γραμμής εντολών. Θα πραγματοποιήσουμε τα πρώτα μας commits και θα συγχρονίσουμε τις αλλαγές μας με το GitHub.

## 2.9 - Δημιουργία GitHub Repository

Θα δημιουργήσουμε το πρώτο μας αποθετήριο στο GitHub και θα οργανώσουμε σωστά το project μας από την αρχή. Θα γνωρίσουμε βασικές έννοιες όπως τα repositories, τα commits και το README.

## 2.10 - FTP / SFTP

Σε αυτό το μάθημα θα μάθουμε πώς μεταφέρονται τα αρχεία μιας ιστοσελίδας από τον υπολογιστή μας στον server. Θα συγκρίνουμε το FTP με το ασφαλέστερο SFTP και θα δούμε πότε χρησιμοποιείται το καθένα.

## 2.11 - FileZilla

Θα εγκαταστήσουμε και θα ρυθμίσουμε το FileZilla ώστε να μπορούμε να ανεβάζουμε και να διαχειριζόμαστε τα αρχεία της ιστοσελίδας μας στον server. Θα πραγματοποιήσουμε τις πρώτες μεταφορές αρχείων και θα γνωρίσουμε τις βασικές λειτουργίες του προγράμματος.

## 2.12 - phpMyAdmin

Θα γνωρίσουμε το phpMyAdmin, το εργαλείο που χρησιμοποιείται για τη διαχείριση βάσεων δεδομένων MySQL μέσω γραφικού περιβάλλοντος. Θα δημιουργήσουμε βάσεις, πίνακες και θα εκτελέσουμε τις πρώτες μας ενέργειες χωρίς χρήση εντολών SQL.

## 2.13 - Image Optimization

Θα μάθουμε πώς να βελτιστοποιούμε εικόνες πριν τις χρησιμοποιήσουμε σε μια ιστοσελίδα. Θα δούμε πώς μειώνουμε το μέγεθος των αρχείων χωρίς αισθητή απώλεια ποιότητας, βελτιώνοντας έτσι την ταχύτητα φόρτωσης και την εμπειρία των επισκεπτών.

# ΕΝΟΤΗΤΑ 3 - Αλγοριθμική Σκέψη

## 3.1 - Τι είναι ένας αλγόριθμος

Πριν γράψουμε οποιαδήποτε γραμμή κώδικα, πρέπει να μάθουμε να λύνουμε προβλήματα. Σε αυτό το μάθημα θα γνωρίσουμε τι είναι ένας αλγόριθμος, πώς περιγράφει μια διαδικασία βήμα προς βήμα και γιατί αποτελεί τη βάση κάθε προγράμματος.

## 3.2 - Flowcharts

Θα μάθουμε να αναπαριστούμε έναν αλγόριθμο με διαγράμματα ροής (Flowcharts). Μέσα από πρακτικά παραδείγματα θα δούμε πώς μπορούμε να οργανώνουμε οπτικά τη λογική ενός προγράμματος πριν ξεκινήσει η υλοποίησή του.

## 3.3 - Ψευδοκώδικας

Σε αυτό το μάθημα θα γνωρίσουμε τον ψευδοκώδικα, έναν απλό τρόπο περιγραφής αλγορίθμων χωρίς τη χρήση συγκεκριμένης γλώσσας προγραμματισμού. Θα μάθουμε να μετατρέπουμε μια ιδέα σε σαφή βήματα που μπορούν εύκολα να υλοποιηθούν σε κώδικα.

## 3.4 - Μεταβλητές

Οι μεταβλητές αποτελούν τον βασικό τρόπο αποθήκευσης πληροφοριών σε κάθε πρόγραμμα. Θα μάθουμε τι είναι, πώς χρησιμοποιούνται και πώς μας επιτρέπουν να διαχειριζόμαστε δεδομένα κατά την εκτέλεση ενός αλγορίθμου.

## 3.5 - Συνθήκες

Θα γνωρίσουμε τις συνθήκες και θα μάθουμε πώς ένα πρόγραμμα μπορεί να λαμβάνει αποφάσεις. Μέσα από παραδείγματα θα χρησιμοποιήσουμε λογικές εκφράσεις ώστε ο αλγόριθμος να ακολουθεί διαφορετική πορεία ανάλογα με τα δεδομένα.

## 3.6 - Βρόχοι

Πολλές φορές χρειάζεται να επαναλάβουμε την ίδια διαδικασία πολλές φορές. Σε αυτό το μάθημα θα μάθουμε τι είναι οι βρόχοι επανάληψης, πότε χρησιμοποιούνται και πώς μπορούν να κάνουν τον κώδικά μας πιο απλό και αποτελεσματικό.

## 3.7 - Συναρτήσεις

Θα μάθουμε πώς χωρίζουμε ένα μεγάλο πρόβλημα σε μικρότερα και πιο εύκολα διαχειρίσιμα τμήματα χρησιμοποιώντας συναρτήσεις. Θα δούμε τα πλεονεκτήματα της επαναχρησιμοποίησης κώδικα και της καλύτερης οργάνωσης των προγραμμάτων μας.

## 3.8 - Επίλυση προβλημάτων

Στο τελευταίο μάθημα της ενότητας θα εφαρμόσουμε όλες τις έννοιες που μάθαμε μέχρι τώρα για να λύσουμε πραγματικά προβλήματα. Στόχος είναι να αναπτύξουμε προγραμματιστική σκέψη και να μάθουμε να σχεδιάζουμε σωστούς αλγορίθμους πριν περάσουμε στην υλοποίηση με κώδικα.

# ΕΝΟΤΗΤΑ 4 - HTML

## 4.1 - Δομή HTML

Σε αυτό το μάθημα θα δημιουργήσουμε την πρώτη μας HTML σελίδα και θα γνωρίσουμε τη βασική δομή κάθε εγγράφου HTML. Θα εξηγήσουμε τον σκοπό των βασικών στοιχείων, όπως τα `<html>`, `<head>` και `<body>`, και θα κατανοήσουμε πώς τα διαβάζει ο browser.

## 4.2 - HTML Tags

Θα γνωρίσουμε τα HTML tags και τον τρόπο με τον οποίο χρησιμοποιούνται για τη δημιουργία μιας ιστοσελίδας. Θα μάθουμε τι είναι τα στοιχεία (elements), τα attributes και πώς γράφουμε σωστό και ευανάγνωστο HTML κώδικα.

## 4.3 - Κείμενα

Θα μάθουμε πώς μορφοποιούμε και οργανώνουμε το περιεχόμενο μιας ιστοσελίδας χρησιμοποιώντας επικεφαλίδες, παραγράφους και άλλα στοιχεία κειμένου. Παράλληλα θα δούμε τη σημασία της σωστής δομής τόσο για τους επισκέπτες όσο και για τις μηχανές αναζήτησης.

## 4.4 - Σύνδεσμοι

Οι σύνδεσμοι αποτελούν τη βάση λειτουργίας του Παγκόσμιου Ιστού. Σε αυτό το μάθημα θα μάθουμε να δημιουργούμε εσωτερικούς και εξωτερικούς συνδέσμους, να ανοίγουμε σελίδες σε νέα καρτέλα και να χρησιμοποιούμε σωστά το στοιχείο `<a>`.

## 4.5 - Εικόνες

Θα προσθέσουμε εικόνες στις ιστοσελίδες μας και θα μάθουμε τις σωστές πρακτικές χρήσης τους. Θα δούμε τη σημασία του εναλλακτικού κειμένου (alt), των σωστών διαστάσεων και της βελτιστοποίησης για καλύτερη απόδοση.

## 4.6 - Lists

Σε αυτό το μάθημα θα δημιουργήσουμε ταξινομημένες, μη ταξινομημένες και σύνθετες λίστες. Θα δούμε πότε χρησιμοποιείται κάθε τύπος λίστας και πώς συμβάλλει στην καλύτερη οργάνωση του περιεχομένου.

## 4.7 - Tables

Θα γνωρίσουμε τους πίνακες HTML και θα μάθουμε πώς παρουσιάζουμε δομημένα δεδομένα. Θα δημιουργήσουμε πίνακες με γραμμές, στήλες και επικεφαλίδες, αποφεύγοντας τη λανθασμένη χρήση τους για τη διάταξη μιας σελίδας.

## 4.8 - Forms

Οι φόρμες επιτρέπουν την αλληλεπίδραση του χρήστη με μια ιστοσελίδα. Θα μάθουμε να δημιουργούμε πεδία εισαγωγής, κουμπιά, λίστες επιλογών και άλλα στοιχεία που χρησιμοποιούνται καθημερινά σε κάθε δυναμική εφαρμογή.

## 4.9 - Semantic HTML

Θα γνωρίσουμε τα σημασιολογικά στοιχεία της HTML και θα μάθουμε πώς κάνουν τον κώδικά μας πιο κατανοητό για τους browsers, τις μηχανές αναζήτησης και τις τεχνολογίες προσβασιμότητας. Θα αντικαταστήσουμε τα γενικά `<div>` όπου χρειάζεται με πιο κατάλληλα στοιχεία.

## 4.10 - Media

Σε αυτό το μάθημα θα ενσωματώσουμε βίντεο, ήχο και άλλο πολυμεσικό περιεχόμενο στις ιστοσελίδες μας. Θα δούμε τους διαθέσιμους τρόπους ενσωμάτωσης και τις βέλτιστες πρακτικές για σωστή συμβατότητα και απόδοση.

## 4.11 - Accessibility

Θα γνωρίσουμε τις βασικές αρχές της προσβασιμότητας (Accessibility) και θα μάθουμε πώς δημιουργούμε ιστοσελίδες που μπορούν να χρησιμοποιηθούν από όλους τους χρήστες, ανεξάρτητα από τις δυνατότητές τους. Θα δούμε πρακτικές που βελτιώνουν τόσο την εμπειρία χρήσης όσο και το SEO.

# ΕΝΟΤΗΤΑ 5 - Bootstrap

## 5.1 - Grid System

Σε αυτό το μάθημα θα γνωρίσουμε το Grid System του Bootstrap, το οποίο μας επιτρέπει να δημιουργούμε γρήγορα responsive διατάξεις. Θα μάθουμε πώς λειτουργούν οι γραμμές, οι στήλες και τα breakpoints, ώστε οι ιστοσελίδες μας να προσαρμόζονται σωστά σε κάθε συσκευή.

## 5.2 - Containers

Θα μάθουμε τι είναι τα containers και γιατί αποτελούν τη βάση κάθε layout στο Bootstrap. Θα γνωρίσουμε τις διαφορές μεταξύ `container`, `container-fluid` και των responsive containers, επιλέγοντας κάθε φορά τη σωστή λύση.

## 5.3 - Navbar

Σε αυτό το μάθημα θα δημιουργήσουμε σύγχρονα και πλήρως responsive μενού πλοήγησης. Θα μάθουμε να χρησιμοποιούμε το Navbar του Bootstrap και να προσαρμόζουμε τη συμπεριφορά του σε κινητές και μεγάλες οθόνες.

## 5.4 - Cards

Οι κάρτες αποτελούν ένα από τα πιο χρήσιμα στοιχεία του Bootstrap. Θα μάθουμε να δημιουργούμε καλαίσθητες παρουσιάσεις περιεχομένου για προϊόντα, άρθρα, υπηρεσίες και πληροφορίες, χρησιμοποιώντας έτοιμα components.

## 5.5 - Buttons

Θα γνωρίσουμε όλα τα διαθέσιμα κουμπιά του Bootstrap και θα μάθουμε πώς να τα προσαρμόζουμε στις ανάγκες της εφαρμογής μας. Θα δούμε διαφορετικά χρώματα, μεγέθη και καταστάσεις, καθώς και καλές πρακτικές σχεδιασμού.

## 5.6 - Forms

Σε αυτό το μάθημα θα δημιουργήσουμε σύγχρονες και καλαίσθητες φόρμες χρησιμοποιώντας τα components του Bootstrap. Θα μάθουμε πώς να οργανώνουμε σωστά τα πεδία εισαγωγής και να βελτιώνουμε την εμπειρία του χρήστη.

## 5.7 - Utilities

Το Bootstrap διαθέτει εκατοντάδες βοηθητικές κλάσεις (Utilities) που μας επιτρέπουν να μορφοποιούμε στοιχεία χωρίς να γράφουμε επιπλέον CSS. Θα γνωρίσουμε τις σημαντικότερες και θα μάθουμε πότε είναι προτιμότερο να χρησιμοποιούνται.

## 5.8 - Bootstrap Icons

Θα μάθουμε πώς να ενσωματώνουμε και να χρησιμοποιούμε τα Bootstrap Icons στις ιστοσελίδες μας. Θα δούμε πώς μπορούν να βελτιώσουν την εμφάνιση και τη χρηστικότητα μιας εφαρμογής χωρίς να επηρεάζουν σημαντικά την απόδοσή της.

## 5.9 - Modal

Σε αυτό το μάθημα θα δημιουργήσουμε αναδυόμενα παράθυρα (Modals) για την εμφάνιση πληροφοριών, φορμών και μηνυμάτων χωρίς αλλαγή σελίδας. Θα μάθουμε πώς λειτουργούν και πότε είναι σωστό να χρησιμοποιούνται.

## 5.10 - Carousel

Θα γνωρίσουμε το Carousel του Bootstrap και θα δημιουργήσουμε παρουσιάσεις εικόνων και περιεχομένου με εναλλαγή διαφανειών. Παράλληλα θα δούμε πότε η χρήση του βελτιώνει την εμπειρία του χρήστη και πότε καλό είναι να αποφεύγεται.

## 5.11 - Offcanvas

Το Offcanvas αποτελεί έναν σύγχρονο τρόπο εμφάνισης πλευρικών μενού και βοηθητικών περιοχών περιεχομένου. Θα μάθουμε πώς δημιουργείται και πώς χρησιμοποιείται κυρίως σε responsive εφαρμογές και κινητές συσκευές.

## 5.12 - Accordion

Σε αυτό το μάθημα θα χρησιμοποιήσουμε το Accordion για την οργάνωση μεγάλου όγκου πληροφοριών σε πτυσσόμενες ενότητες. Θα δούμε πώς βελτιώνει την εμπειρία του χρήστη και σε ποιες περιπτώσεις αποτελεί την ιδανική επιλογή.

# ΕΝΟΤΗΤΑ 6 - CSS

## 6.1 - Selectors

Θα γνωρίσουμε τους βασικούς selectors της CSS, ώστε να μπορούμε να τροποποιούμε ή να επεκτείνουμε τα styles του Bootstrap μέσα από το δικό μας αρχείο `custom.css`.

## 6.2 - Colors & Units

Θα μάθουμε τους βασικούς τρόπους ορισμού χρωμάτων και τις σημαντικότερες μονάδες μέτρησης της CSS, ώστε να μπορούμε να προσαρμόζουμε εύκολα την εμφάνιση μιας εφαρμογής.

## 6.3 - Box Model

Θα κατανοήσουμε πώς λειτουργούν τα margin, padding και border, ώστε να μπορούμε να διορθώνουμε αποστάσεις και στοίχιση στοιχείων όταν χρειάζεται.

## 6.4 - Display & Position

Θα γνωρίσουμε τις βασικές ιδιότητες `display` και `position`, οι οποίες χρησιμοποιούνται συχνά για μικρές διορθώσεις και ειδικές περιπτώσεις διάταξης.

## 6.5 - Flexbox

Θα γνωρίσουμε τις βασικές αρχές του Flexbox, ώστε να κατανοούμε καλύτερα τη λειτουργία του Bootstrap και να μπορούμε να επεμβαίνουμε όπου χρειάζεται.

## 6.6 - Responsive Design

Θα μάθουμε τις βασικές αρχές του Responsive Design και τη χρήση των media queries για ειδικές περιπτώσεις που δεν καλύπτονται από το Bootstrap.

## 6.7 - Custom CSS

Στο τελευταίο μάθημα της ενότητας θα δημιουργήσουμε ένα οργανωμένο `custom.css` αρχείο και θα μάθουμε πώς προσαρμόζουμε με ασφάλεια το Bootstrap χωρίς να τροποποιούμε τα αρχικά του αρχεία.

## 7.2 - Variables

Οι μεταβλητές αποτελούν τον βασικό μηχανισμό αποθήκευσης δεδομένων στην PHP. Θα μάθουμε πώς δηλώνονται, πώς χρησιμοποιούνται και πώς μπορούμε να αποθηκεύουμε και να επεξεργαζόμαστε διαφορετικούς τύπους πληροφοριών.

## 7.3 - Constants

## 8.14 - Indexes

Στο τελευταίο μάθημα της ενότητας θα γνωρίσουμε τους δείκτες (Indexes) και τον σημαντικό ρόλο τους στην απόδοση μιας βάσης δεδομένων. Θα μάθουμε πώς επιταχύνουν τις αναζητήσεις, πότε πρέπει να δημιουργούνται και πότε η υπερβολική χρήση τους μπορεί να έχει αρνητικές συνέπειες.

# ΕΝΟΤΗΤΑ 10 - Ασφάλεια & Δημοσίευση

## 10.1 - Security

Σε αυτό το μάθημα θα γνωρίσουμε τις βασικές αρχές ασφάλειας που πρέπει να εφαρμόζονται σε κάθε web εφαρμογή. Θα δούμε τις συνηθέστερες απειλές, τα πιο κοινά λάθη των προγραμματιστών και τις πρακτικές που θα ακολουθούμε σε όλη τη συνέχεια της σειράς.

## 10.2 - SQL Injection

Το SQL Injection αποτελεί μία από τις πιο γνωστές επιθέσεις σε εφαρμογές που χρησιμοποιούν βάσεις δεδομένων. Θα κατανοήσουμε πώς πραγματοποιείται, ποιες είναι οι συνέπειές του και πώς μπορούμε να προστατεύσουμε αποτελεσματικά τις εφαρμογές μας.

## 10.3 - XSS

Σε αυτό το μάθημα θα γνωρίσουμε τις επιθέσεις Cross-Site Scripting (XSS). Θα δούμε πώς ένας επιτιθέμενος μπορεί να εισάγει κακόβουλο κώδικα σε μια ιστοσελίδα και ποιες τεχνικές χρησιμοποιούνται για την αποτροπή τέτοιων επιθέσεων.

## 10.4 - CSRF

Θα μάθουμε τι είναι οι επιθέσεις Cross-Site Request Forgery (CSRF) και γιατί αποτελούν σοβαρό κίνδυνο για τις web εφαρμογές. Θα υλοποιήσουμε μηχανισμούς προστασίας ώστε να αποτρέπουμε μη εξουσιοδοτημένες ενέργειες εκ μέρους των χρηστών.

## 10.5 - Prepared Statements

Τα Prepared Statements αποτελούν τον ασφαλέστερο τρόπο εκτέλεσης SQL εντολών. Θα μάθουμε πώς χρησιμοποιούνται με το PDO και γιατί αποτελούν την κύρια άμυνα απέναντι στις επιθέσεις SQL Injection.

## 10.6 - Validation

Σε αυτό το μάθημα θα μάθουμε να ελέγχουμε την εγκυρότητα των δεδομένων που εισάγει ο χρήστης. Θα εφαρμόσουμε κανόνες ελέγχου τόσο στην πλευρά του browser όσο και στον διακομιστή, διασφαλίζοντας την ορθότητα των πληροφοριών.

## 10.7 - Sanitization

Η σωστή επεξεργασία των δεδομένων είναι απαραίτητη για την ασφάλεια μιας εφαρμογής. Θα μάθουμε πώς καθαρίζουμε τις εισόδους των χρηστών και πώς αποτρέπουμε την εκτέλεση κακόβουλου περιεχομένου πριν αυτό αποθηκευτεί ή εμφανιστεί.

## 10.8 - SSL

Θα γνωρίσουμε το πρωτόκολλο HTTPS και τα πιστοποιητικά SSL/TLS. Θα μάθουμε γιατί είναι απαραίτητα για την προστασία των δεδομένων, πώς λειτουργεί η κρυπτογράφηση και πώς ενεργοποιούμε ένα πιστοποιητικό σε έναν server.

## 10.9 - Deployment

Σε αυτό το μάθημα θα δημοσιεύσουμε για πρώτη φορά την εφαρμογή μας σε πραγματικό server. Θα δούμε όλα τα απαραίτητα βήματα ώστε το project να μεταφερθεί σωστά από το τοπικό περιβάλλον στο Internet.

## 10.10 - Backups

Θα μάθουμε γιατί τα αντίγραφα ασφαλείας είναι απαραίτητα σε κάθε έργο. Θα δούμε πώς δημιουργούνται backups των αρχείων και της βάσης δεδομένων, καθώς και στρατηγικές για γρήγορη επαναφορά σε περίπτωση προβλήματος.

## 10.11 - Cron Jobs

Σε αυτό το μάθημα θα γνωρίσουμε τα Cron Jobs και θα μάθουμε πώς αυτοματοποιούμε επαναλαμβανόμενες εργασίες, όπως αποστολή email, δημιουργία αντιγράφων ασφαλείας και προγραμματισμένες ενημερώσεις της εφαρμογής.

## 10.12 - Performance

Θα γνωρίσουμε βασικές τεχνικές βελτιστοποίησης της απόδοσης μιας web εφαρμογής. Θα δούμε πώς μειώνουμε τον χρόνο φόρτωσης, βελτιώνουμε την ταχύτητα εκτέλεσης και προσφέρουμε καλύτερη εμπειρία στους χρήστες.

## 10.13 - Caching

Το Caching αποτελεί μία από τις σημαντικότερες τεχνικές βελτίωσης της απόδοσης. Θα μάθουμε πώς λειτουργεί, ποια είδη cache υπάρχουν και πώς μπορούμε να μειώσουμε σημαντικά τον φόρτο του server.

## 10.14 - SEO

Σε αυτό το μάθημα θα γνωρίσουμε τις βασικές αρχές του Search Engine Optimization (SEO). Θα μάθουμε πώς δημιουργούμε ιστοσελίδες που κατανοούνται καλύτερα από τις μηχανές αναζήτησης και αυξάνουν τις πιθανότητες εμφάνισής τους στα αποτελέσματα αναζήτησης.

## 10.15 - Analytics

Στο τελευταίο μάθημα της ενότητας θα μάθουμε πώς παρακολουθούμε την επισκεψιμότητα και τη συμπεριφορά των χρηστών μέσω εργαλείων ανάλυσης, όπως το Google Analytics. Θα δούμε πώς αξιοποιούμε τα δεδομένα αυτά για τη συνεχή βελτίωση της ιστοσελίδας και της εφαρμογής μας.
